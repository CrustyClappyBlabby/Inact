# Inact – Henning's Family Dashboard

A React + TypeScript app that visualises Henning's family data:

1. **Family tree** – interactive expand/collapse tree view
2. **Age bar chart** – ages of every family member
3. **Average shoe size per gender** – table + bar chart
4. **Lars growth & forecast** – measured heights with a linear-regression forecast to age 18

## Prerequisites

- [Node.js](https://nodejs.org/) 18 or newer (includes `npm`)

## Install dependencies

```bash
npm install
```

## Run the development server

```bash
npm start
```

Then open <http://localhost:1234> in your browser. The page hot-reloads on every file save.

## Build for production

```bash
npm run build
```

Output is written to the `dist/` folder.
